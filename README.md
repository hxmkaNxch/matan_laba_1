### описание проекта:

лабораторная работа по анализу динамических систем с дискретным временем. исследуются свойства логистического отображения и модифицированного отображения варианта 4.

### структура проекта:

- `lab1_matan.py` - основной код с анализом (easy + normal levels)
- `proofs.md` - теоретические выкладки и доказательства
- `requirements.txt` - зависимости python

### выполненные задания:

#### easy level:
- доказательство сохранения интервала для логистического отображения
- сравнительный анализ логистического отображения и варианта 4
- построение графиков зависимости $x_{n+1}$ от $x_n$

#### normal level:
- анализ неподвижных точек обоих отображений
- исследование сходимости к нулю
- анализ монотонности подпоследовательностей
- построение графиков последовательностей $x_n$ от $n$

#### hard level:
- анализ устойчивости неподвижных точек
- исследование циклов различных порядков
- построение лестниц ламерея
- создание бифуркационных диаграмм
- сравнительный анализ динамики обоих отображений

#### expert level:
- полный анализ устойчивости неподвижных точек
- исследование чувствительности к начальным условиям (эффект бабочки)
- построение детальных бифуркационных диаграмм с окнами периодичности
- визуализация фрактальной структуры бифуркационного дерева
- теоретические доказательства устойчивости и сходимости
- анализ хаотического поведения и окон периодичности
- сравнение динамики логистического отображения и варианта 4 в хаотическом режиме
